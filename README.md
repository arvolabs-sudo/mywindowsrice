# My Windows Setup Configs

This repository contains the configuration files and customizations I use for my personal Windows setup.

The goal of this setup is to create a cleaner, minimal, and more personalized Windows experience inspired by Linux desktop environments.

## Included

* YASB configuration
* GlazeWM configuration
* Windows Terminal settings
* Oh My Posh configuration
* Fastfetch configuration
* Other customization files

## Important

These configs are made for my personal setup, so you may need to modify paths, settings, or preferences to match your system.

Before applying system-level customizations, I recommend creating a restore point.
